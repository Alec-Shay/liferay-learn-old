Shipping Methods¶

Liferay Commerce supports several shipping methods and carriers. Before choosing any shipping method, verify the Measurement Units that the store will use.

Here are several shipping methods:

  • Variable Rate

  • Flat Rate

  • FedEx

    Note that FedEx is available only for Commerce Enterprise subscribers. For users interested in adding other carriers, see Implementing a New Shipping Engine.

Shipping methods may be restricted by country or region. To learn more, see Applying Shipping Method Restrictions.
