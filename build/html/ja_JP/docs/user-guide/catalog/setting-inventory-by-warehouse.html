Setting Inventory by Warehouse¶

This article documents how to set inventory by warehouse. Some stores may have more than one warehouse and store managers can set product quantity per location. The Inventory menu is where store administrators manage the number of products available for sale in each warehouse. If the Minium Accelerator was used to create the store, there are three warehouses already created.

To set inventory by warehouse:

  1. Navigate to the Control Panel → Commerce → Products.

  2. Click on a product (for example U-Joint).

  3. Click the SKUs sub-tab.

  4. Click on the SKU ID (MIN-55861).

  5. Click Inventory on the left menu.

  6. Enter a quantity for the corresponding warehouse(s).

  7. Click Save for each warehouse.

  8. Repeat these steps for all the other product variants (SKUs) associated for a product.

Each warehouse now has the correct quantity available for sale.
